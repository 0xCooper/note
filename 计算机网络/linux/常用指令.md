### 文件挂载

### yum源得配置

由于载安装得时候并没有配置gcc ,但是载ios中是存在gcc得就直接配置yum源为cdrom

```bash
  cd /mnt
   mkdir cdrom
  mount /dev/cdrom /mnt/cdrom
  vim /etc/yum.repos.d/dvd.repo
  *************
  [dvd]
  name=dvd
  baseurl=file:///mnt/cdrom
  enabled=1
  gpgcheck=0
  *************
   yum install gcc
 yum intsall system-config-lvm
#方法很简单而且ios中有nmap与wireshark与gcc gimp
```
