1，使用new 分配内存

```cpp
int *ptr_int =new int;
delete ptr_int;//释放由new分配的内存与C中的free一样
```

```cpp
int *nums =new int[5];
//运行到这段话的时候分配给了一个空间
```

```cpp
int num[5];
int*nums= new int[5];
cout<<sizeof(num)<<sizeof(num)<<endl;
//new 分配的空间是在堆里
```
