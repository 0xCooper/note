1.认识php

---

1.1 echo

```
<?php echo "Hello world!";?>
```

1.2计算表达式

```
<?php echo 12*3 ?>
```

1.3字符串

```
<?php echo 'Hi,'.'imooc!';?>
```

在php中字符串**连接符**是用点（.）来表示的，这一点比较特殊

这点注意，在PHP编程中需要在每条语句的末尾加入分号`；`

1.4注释
在PHP中也有注释语句：用双斜杠（//）来表示。其它语言中，Html中使用<!--注释语句-->，CSS中使用/*注释语句*/。
