##  # 实现每天自动上传



新建一个git.bat文件，复制下面的代码，记得修改路径

```bash
@echo off
@title bat 交互执行git命令
D:
cd D:/git/test
git add .
git commit -m %date:~0,4%年%date:~5,2%月%date:~8,2%日


```
